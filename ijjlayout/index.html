ijjAlertBox.js

—— 原生js弹框插件

注：引入Bootstrap.min.css仅提供index.html页面内容显示,无其他作用

Open

点击按钮查看效果 
ijjAlertBox.confirm({
    content: 'Sed ut perspiciatis ......',
    okBtn: '好的',
    contentColor: 'red',
    onConfirm: function () {
        console.log("OK - 回调触发后隐藏提示框");
        alert("OK - 回调触发后隐藏提示框");
    },
    onCancel: function () {
        console.log("Cancel-回调触发后隐藏提示框");
        alert("Cancel-回调触发后隐藏提示框");
    }
});

使用说明

  • 1. 引入js 和 css 文件
  • 2. 通过全局对象 ijjAlertBox 调用相关方法两个方法为： ijjAlertBox.open({}); ijjAlertBox.colse({});

    • 配置参数

    参数属性 意义 适用方法 是否必须
    title 弹框标题 alert、confirm 否，默认 : 提示信息
    content 提示内容主体 alert、confirm 否，默认 : 提示内容
    contentColor 提示内容颜色 alert、confirm 否，默认 : #333333
    okBtn 确认按钮文字 alert、confirm 否，默认 : 好的
    promptOkBtn prompt确认按钮文字 prompt 否，默认 : 确认
    okBtnColor 确认按钮颜色 alert、confirm 否，默认 : #0e90d2
    cancelBtn 取消按钮文字 alert、confirm 否，默认 : 取消
    onConfirm 确认按钮事件 alert、confirm 否，默认 : 不触发事件
    onCancel 取消按钮事件 alert 否，默认 : 不触发事件
